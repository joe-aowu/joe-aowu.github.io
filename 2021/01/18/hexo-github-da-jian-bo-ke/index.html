hexo+github搭建博客

hexo 博客
hexo
发布日期:   2021-01-18
文章字数:   478
阅读次数:  

生成ssh密钥

git配置信息：

git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"

git生成ssh密钥：

ssh-keygen -t rsa -C "你的GitHub注册邮箱"

输入三个回车，

在文件夹中找到id_rsa.pub文件，用记事本打开，复制内容

在github中进行ssh的设置，点击设置

输入title（随意），将刚才复制的内容粘贴到Key

在git bash中输入ssh git@github.com检测是否设置成功

输入yes

出现successfully即可

安装node.js

参考博客

安装过程可能出现的错误

错误1：ERR code ENOTFOUND

解决方法：

npm install -g cnpm --registry=https://registry.npm.taobao.org

之后继续刚刚的install

错误2：ERR code EAI_AGAIN

解决方法：换一个网（我从wifi换到手机热点即可）

github端操作

在github新建一个repository，Repository name 输入 用户名.github.io

hexo搭建

进入文件夹，在命令行或者右键git bash输入

npm install -g hexo-cli

ps：该过程可能会很慢，还可能会报错

若出现报错，尝试使用

npm cache clean -f

清理npm缓存，再次进行install，这次就不慢了

之后进行初始化和安装

hexo init   #初始化
npm install #安装

在git bash使用

hexo g

生成页面

使用

hexo s

预览页面

此时访问 http://localhost:4000 出现页面即为成功

在bash

ctrl+c

停止预览

安装 hexo-deployer-git：

npm install hexo-deployer-git --save

然后修改 _config.yml 文件末尾的 Deployment 部分，修改成如下：

deploy:
  type: git
  repository: https://用户名.github.io/
  branch: main

github默认分支改为main了，大部分教程的branch还是master，注意一下就好

部署

hexo d

輸入地址 https://用户名.github.io/ 即可访问到你的博客

接下来就可以挑选一个喜欢的主题，根据相应教程对你的博客进行更改( > ̀◡•́ )✧

传送门奉上：Themes | Hexo
